This directory contains a few sample data sets for BOXER, and scripts
illustrating how to feeed them into BOXER applications (BORJ,
Repeater, and WindowRepeater)

Subdirectories:

suites: suite files

datasets: dataset files

learner: learner definition files

priors: priors definition files

scripts: sample scripts

orthogonal: a very simple and easy data set (just 3 examples), with scripts

SimpleDataSet: another pretty simple, linearly separable data set, with scripts

letter-recognition: scripts for the Letter Recognition data set (
http://archive.ics.uci.edu/ml/datasets/Letter+Recognition )
