A standalone Java program that runs machine learning experiments using BOXER.

This package includes the class {@link edu.dimacs.mms.borj.Driver}, which is the main class for a small command-line-driven application that uses BOXER, as well as a number of auxiliary classes.

One can use BORJ as an independent application, providing command-line interface to most of BOXER's essential functionality. You may also use borj.Driver as a model for your application.

For usage details, please read the documentation for the class {@link edu.dimacs.mms.borj.Driver}.
