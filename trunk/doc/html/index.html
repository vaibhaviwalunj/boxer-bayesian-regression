BOXER and BORJ documentation - main page

This is the main page for the documentation distributed with the BOXER project software. Software produced by the project includes BOXER (a Java library for online, anytime learning of polytomous logistic regression models), as well as a number of auxiliary tool. Those include BORJ (Bayesian Online Regression in Java), a standalone Java program that runs machine learning experiments using BOXER.
